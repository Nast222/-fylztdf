# Training Planner

**Автор:** [Бандяева]
**Язык:** Python 3.x + Tkinter

## Описание

Графическое приложение для планирования тренировок. Позволяет добавлять записи о тренировках (дата, тип, длительность), фильтровать их по типу и дате, а также сохранять и загружать данные в формате JSON.

## Возможности

- Добавление новых тренировок с проверкой корректности ввода.
- Фильтрация по типу и дате.
- Сохранение данных в `data.json`.
- Простой и интуитивный интерфейс.

## Использование

1. Запустите `training_planner.py`.
2. Введите данные о тренировке и нажмите «Добавить тренировку».
3. Для фильтрации используйте поля в блоке «Фильтр».
4. Данные сохраняются автоматически при добавлении новой записи.
